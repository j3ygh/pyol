win+R > cmd

to install:

    cd /your/python/env/path
    git clone https://github.com/j3ygithub/pyol
    python -m venv pyol
    cd pyol
    Scripts\activate
    pip install -r requirements.txt

to make a mail job:

    edit /your/python/env/path/pyol/pyol/main.py

to run it:

    cd /your/python/env/path/pyol/pyol/
    python main.py
